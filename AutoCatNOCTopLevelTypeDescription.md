# NOC Ticket Categorization Guidelines

**Categorize as "NOC" if a ticket involves any of the following:**

- **Connectivity or Outage Issues:**  
  - Tickets reporting network downtime, intermittent connectivity, or degraded performance affecting users.

- **Hardware or Software Failures:**  
  - Issues involving malfunction or failure of network equipment (e.g., routers, switches, firewalls) or critical servers/applications.

- **Performance and Capacity Issues:**  
  - Reports of network congestion, high latency, packet loss, or bandwidth bottlenecks impacting performance.

- **Change and Configuration Management:**  
  - Requests for network configuration updates, software patches, or planned changes that require coordination to maintain network stability.

- **Maintenance and Scheduled Tasks:**  
  - Notifications related to routine maintenance, hardware upgrades, or pre-scheduled downtime for system optimization.

**Examples:**

- "Users experiencing intermittent connectivity and degraded network performance during peak hours."
- "Alert: Critical server downtime due to a network outage impacting multiple departments."
- "Request to update network configuration and apply necessary patches during the upcoming maintenance window."
